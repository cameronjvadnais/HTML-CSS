Definition

Here are some terms to know for css.

A rule would be considered: p {…}

The selector in this case is: p

A rule is made up of selectors and declarations.

A declaration is: {property:value}

The entire rule would be: selector { property:value }

A rule can have multiple declarations and multiple selectors so we can actually have: selector, selector2 { property:value; property2:value; }

A rule set would be multiple rules.

Further HTML definitions.

Style can be a tag and an attribute. The style tag would go into the head tag.
