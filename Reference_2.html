HTML/CSS definitions

The difference between elements and tags.

The difference between elements and tags is this. HTML tag is just opening or closing entity.

HTML element encompasses opening tag, closing tag, and content.

Opening CSS using style tag or head tag.

You can use the style tag in the head of the HTML. Use this tag when you don't want to link to a seperate CSS page. Use the link rel="stylesheet" type="text/css" href="text.css" tag in the head to link to a seperate CSS page.

  • HREF: HREF stands for Hypertext REFerence.
  • REL : REL stands for RELationship.

Further Definitions and References.

  • Attribute: A attribute alters the element. An attribute is placed inside of the opening tag of a tag.
  • Hypertext: Text with links in it. Hypertext is text which contains links to other texts. Any time you click on a word that brings you to a new webpage, you've clicked on hypertext.
  • Markup Language: A programming language used to make text do more than just sit on a page: it turns text into images, links, tables, lists, and much more. HTML is the markup language I'm using.
  • Nesting: To place one tag/element inside another one.
